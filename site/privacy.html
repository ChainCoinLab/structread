StructRead Privacy Policy

Last updated: April 21, 2026

Overview

StructRead is a Chrome extension that helps users understand complex English sentences by analyzing selected text, highlighting grammatical structure, and saving vocabulary items for later review.

This policy explains what data the product processes, how it is used, and when it may be shared.

Data We Process

  • Text that a user explicitly selects for sentence analysis
  • Words that a user explicitly selects for word lookup
  • Vocabulary items saved by the user
  • Extension preferences such as reading mode, highlight colors, API endpoint settings, and provider settings
  • API provider settings entered by the user, such as model selection and API key values

How We Use Data

  • Analyze selected English sentences
  • Return sentence structure explanations
  • Highlight sentence components on the current page
  • Look up word definitions and pronunciation data
  • Save vocabulary for later review or export
  • Remember extension settings and preferred providers

Data Sharing

Selected text or selected words may be transmitted to the following services when required to provide the feature you requested:

  • The developer backend service that performs or coordinates sentence and word analysis
  • Third-party AI model providers that the user explicitly configures for analysis

StructRead does not sell user data, does not use user data for advertising or profiling, and does not transfer user data to data brokers or information resellers.

Local Storage

The extension stores some information locally in Chrome storage, including:

  • Saved vocabulary
  • Highlight color preferences
  • Reading mode selection
  • Provider configuration
  • Active provider selection
  • Endpoint configuration

Human Access

Humans do not read user-selected text except when necessary for security, legal compliance, or user-requested support involving specific data.

Data Retention

Locally stored data remains in the user's browser until the user deletes it, clears extension storage, or uninstalls the extension.

Server-side retention, if any, should be limited to what is necessary to provide the service securely and reliably.

Security

Production deployments should use HTTPS for all extension-to-server communication. API keys and any transmitted text should be handled securely and only for the extension's stated purpose.

User Choices

  • Choose whether to analyze selected text
  • Choose whether to save vocabulary
  • Remove saved vocabulary from local storage
  • Export saved vocabulary
  • Configure or remove AI providers
  • Uninstall the extension at any time

Chrome Web Store Disclosure

The use of data processed by this extension is limited to providing and improving its user-facing English reading and sentence analysis features, in line with the Chrome Web Store User Data Policy.

StructRead helps readers understand English by structure.
