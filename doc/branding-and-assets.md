# Branding And Assets

This document defines the recommended product naming direction and the first batch of Chrome Web Store asset briefs.

## Recommended Product Name

`StructRead`

Why this is the recommended choice:

- directly expresses `structure + reading`
- compared with generic analyzer-style names, it is more productized
- matches the core value of AI sentence parsing and visual structure understanding
- suitable for later expansion into vocabulary, courses, and personalized learning

## Recommended Store Title

`StructRead`

## Recommended Store Summary

`Understand English instantly with AI-powered sentence parsing and structure visualization.`

## Recommended Tagline

`Understand English by Structure`

## Alternative Names

### Strong alternatives

- `StructRead AI`
- `ParseRead`
- `Sentence Lens`
- `Syntax Lens`

### Avoid these directions

- overly technical names centered on `analyzer`, `parser`, or `grammar engine`
- names that sound like a school exam tool only
- names that are too abstract and do not suggest reading help

## Positioning

Primary positioning:

- AI English reading assistant for non-native English speakers

Secondary positioning:

- grammar pattern learning from real-world articles
- vocabulary collection during reading

## Audience

- non-native English readers
- students reading essays, news, and exam passages
- professionals reading English content for work

## Visual Direction

Core visual idea:

- make sentence structure feel visible

Design cues:

- sentence fragments highlighted in clean blocks
- blue / coral / green core-role colors
- subtle gray for weak modifiers
- a calm academic tone, not childish
- light background with strong contrast

## Chrome Web Store Assets

### 1. Store icon

Required size:

- `128x128`

Concept:

- a stylized sentence line
- three highlighted segments representing subject / verb / object
- simple geometric form
- avoid tiny letters that disappear at small sizes

### 2. Screenshot set

Required size:

- `1280x800` preferred

Recommended set:

1. Real article page with selected sentence before analysis
2. Highlighted sentence showing subject / verb / object
3. Learning mode with explanation panel
4. Word lookup popup with saved vocabulary
5. Settings panel with modes, colors, and AI providers

### 3. Small promo tile

Required size:

- `440x280`

Headline:

`Understand English by Structure`

Visual:

- one long sentence broken into colored segments
- small explanation card on the right
- minimal text

### 4. Marquee image

Optional size:

- `1400x560`

Headline options:

- `Understand Long English Sentences Faster`
- `Read English by Structure, Not Guesswork`

## Image Generation Prompts

These prompts are intended for AI image generation or as briefs for a designer.

### Icon prompt

`Create a clean 128x128 app icon for a Chrome extension named StructRead. Show a minimal sentence line broken into three colored segments representing subject, verb, and object. Use a modern flat style, crisp geometry, strong contrast, light background, and no tiny text. The icon should remain recognizable at small sizes.`

### Small promo tile prompt

`Create a 440x280 Chrome Web Store promo tile for a reading extension named StructRead. Show one long English sentence visually split into colored structure blocks, with a compact explanation card floating beside it. Use a light editorial interface style, blue coral and green accent colors, subtle gray support colors, minimal text, and clean professional composition. Include the headline "Understand English by Structure".`

### Marquee prompt

`Create a 1400x560 marquee image for a Chrome extension named StructRead. The composition should show complex English text transformed into clear colored sentence structure, with a refined educational product aesthetic. Use a bright background, editorial layout, strong hierarchy, and restrained text. Headline: "Understand Long English Sentences Faster".`
